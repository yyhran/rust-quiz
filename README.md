# rustquiz

 ## Attribution Notice
The quiz questions are derived from the [Rust Quiz](https://github.com/dtolnay/rust-quiz).

This application's interface design is based on the UI structure of [CppQuiz](https://github.com/vsklamm/CppQuiz).
